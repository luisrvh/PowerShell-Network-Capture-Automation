# PowerShell-Network-Capture-Automation
Herramienta creada para capturar y analizar rápidamente tráfico de red en PCs corporativas donde no se permite instalar Wireshark por estar en producción. Usa utilidades nativas de Windows para generar trazas, guardarlas localmente y después revisarlas desde otra PC con Wireshark. Ideal para diagnóstico e incidentes.
